# Unity UI Polish

Inspect whether the project uses UI Toolkit, uGUI, or both. Match it.

Design with:
- a consistent spacing scale and typography hierarchy
- anchors or responsive layout rules
- safe areas and touch target sizes
- loading, empty, error, disabled, hover, pressed, selected, and success states
- readable contrast and scalable text
- keyboard/controller navigation where relevant
- reduced-motion accommodation where supported
- no clipped panels, overlapping labels, or off-screen modal actions

Validate 16:9, 16:10, ultrawide, tablet, and representative phone aspect ratios
when those platforms are supported.
