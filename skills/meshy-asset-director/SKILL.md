# Meshy Asset Director

Create an asset brief before generation:
purpose, style, silhouette, dimensions, camera distance, poly budget, texture
requirements, rig type, animation list, collider plan, and acceptance criteria.

Choose:
- Text-to-3D for concept-led assets
- Image-to-3D for a strong reference
- Multi-image for coherent multiple views
- Remesh for topology or polygon problems
- Retexture for surface/style problems
- Rigging for suitable humanoid bipeds
- Animation only after successful rig review

Before every paid call:
1. authenticate
2. check balance
3. describe the exact operation
4. obtain explicit approval
5. submit with `--confirm-spend`
6. preserve task IDs
7. download outputs promptly
8. record sanitized metadata

Reject fused limbs, occluded joints, duplicate geometry, severe holes, unusable
topology, or silhouettes that fail at gameplay distance.
